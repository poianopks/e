1 파이썬(Python)이란?

1

초보자에게 좋은 언어

파이썬은 영어 문법처럼 자연스러운 프로그래밍 언어입니다. 배우기 쉽고 작성하기 편리해서 전 세계에서 가장 인기 있는 언어 중 하나입니다.

2

다양한 분야에서 사용

웹 개발, 데이터 분석, 인공지능, 자동화, 게임 개발 등 거의 모든 분야에서 사용됩니다. 특히 시작하기 좋은 언어입니다.

3

간결하고 강력함

다른 언어에 비해 코드가 간결하고 가독성이 높습니다. "Hello, World!" 출력이 한 줄로 가능할 정도로 간단합니다.

print("안녕하세요, 파이썬!")

2 Flask란 무엇인가?

4

웹 개발을 위한 도구

Flask는 파이썬으로 웹사이트를 만들 수 있게 해주는 작고 강력한 도구입니다. 마이크로 웹 프레임워크라고 불립니다.

5

간단하고 유연함

복잡한 설정 없이도 웹 애플리케이션을 빠르게 만들 수 있습니다. 필요한 기능만 골라서 사용할 수 있어서 초보자에게 적합합니다.

6

무엇을 만들 수 있나?

블로그, 쇼핑몰, API 서버, 데이터 대시보드 등 거의 모든 종류의 웹 애플리케이션을 만들 수 있습니다.

3 Flask 설치와 기본 구조

7

Flask 설치하기

명령 프롬프트나 터미널에서 다음 명령어를 입력합니다:

pip install flask

팁: pip는 파이썬 패키지 설치 도구로, 파이썬을 설치하면 자동으로 포함됩니다.

8

기본 구조 시작하기

Flask 애플리케이션은 다음과 같은 기본 구조로 시작합니다:

from flask import Flask

# Flask 애플리케이션 생성
app = Flask(__name__)

# 루트 경로(/)에 접속했을 때 실행할 함수
@app.route('/')
def home():
  return "안녕하세요, Flask!"

# 애플리케이션 실행
if __name__ == '__main__':
  app.run(debug=True)
9

코드 설명

  • from flask import Flask: Flask 모듈 가져오기
  • app = Flask(__name__): Flask 애플리케이션 생성
  • @app.route('/'): 웹 주소(/)에 접속했을 때 실행할 함수 지정
  • def home():: 실제로 실행할 함수 정의
  • app.run(debug=True): 서버 실행 (디버그 모드 활성화)

4 Flask로 HTML 만들기

10

render_template_string

Flask에서 HTML을 사용하려면 render_template_string 함수를 사용합니다:

from flask import render_template_string

@app.route('/')
def home():
  html_content = '''
<html>
<head><title>나의 첫 웹페이지</title></head>
<body>
<h1>안녕하세요!</h1>
<p>이것은 Flask로 만든 웹페이지입니다.</p>
</body>
</html>'''
  return render_template_string(html_content)
11

HTML 기본 구조

모든 HTML 문서는 기본적으로 다음 구조를 가집니다:

<!DOCTYPE html>
<html>
<head>
  <title>페이지 제목</title>
</head>
<body>
  <!-- 내용 작성 -->
</body>
</html>
12

자주 사용하는 HTML 태그

  • <h1> ~ <h6>: 제목 (숫자가 작을수록 큰 제목)
  • <p>: 문단
  • <a href="...">: 링크
  • <img src="...">: 이미지
  • <div>: 영역 구분
  • <style>: CSS 스타일

5 Flask 애플리케이션 실행

13

파일 저장하기

코드를 app.py 파일로 저장합니다. (다른 이름도 가능하지만 app.py가 일반적)

주의: 파일 인코딩은 UTF-8로 저장해야 한글이 정상적으로 표시됩니다.

14

터미널에서 실행

파일이 저장된 폴더에서 터미널을 열고 다음 명령어를 입력합니다:

python app.py

성공적으로 실행되면 다음과 같은 메시지가 표시됩니다:

* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
* Restarting with stat
* Debugger is active!
15

웹 브라우저에서 확인

웹 브라우저를 열고 주소창에 다음을 입력합니다:

http://localhost:5000

또는

http://127.0.0.1:5000

만든 웹페이지가 표시되는지 확인합니다.

6 디버깅과 팁

16

디버그 모드

개발 중에는 debug=True를 설정하는 것이 좋습니다:

app.run(debug=True)
  • 에러 발생 시 상세한 정보 표시
  • 코드 수정 후 자동으로 서버 재시작
  • 실제 서비스 환경에서는 반드시 False로 변경
17

자주 발생하는 오류

  • 들여쓰기 오류: 파이썬은 들여쓰기가 매우 중요합니다
  • 괄호 불일치: (), {}, [], '', "" 항상 쌍으로 맞추기
  • 한글 깨짐: 파일 인코딩을 UTF-8로 설정
  • 포트 충돌: 다른 프로그램이 5000번 포트를 사용 중일 때
18

🚨 흔한 실수 3가지

  • 들여쓰기 오류: 파이썬은 띄어쓰기 4칸 필수!

    해결법: 모든 블록(함수, 조건문, 반복문)은 4칸 들여쓰기로 시작해야 합니다.

    def hello():
    # 4칸 들여쓰기
    return "Hello"
  • 괄호 불일치: ( ), { }, [ ], ' ' 항상 쌍으로

    해결법: 열린 괄호는 항상 닫힌 괄호와 짝을 이루어야 합니다.

    # 올바른 예
    app = Flask(__name__)

    # 잘못된 예 (닫는 괄호 누락)
    app = Flask(__name__ # 오류 발생!
  • 한글 깨짐: 파일 인코딩 → 반드시 UTF-8 사용

    해결법: 코드 편집기에서 파일을 저장할 때 인코딩을 UTF-8로 설정합니다.

    # 파일 맨 위에 인코딩 선언 추가
    # -*- coding: utf-8 -*-
