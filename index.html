🎯 Flask 완전 초보 가이드 (18단계)

1 파이썬(Python)이란?

▶ 사람 말처럼 쉽게 배울 수 있는 프로그래밍 언어
▶ 웹 개발, 데이터 분석, 인공지능 등에 사용

2 Flask란?

▶ 파이썬으로 웹사이트를 만드는 도구상자
▶ 레고 조립하듯 간단하게 웹앱 제작 가능

3 필수 설치

▶ 파이썬 설치: 공식사이트
▶ 명령창에서: pip install flask

4 기본 구조 시작

from flask import Flask
app = Flask(__name__)

▶ Flask 도구 가져오기
▶ 'app'이 우리 웹앱의 이름

5 라우팅(Route)

@app.route('/')

▶ "사용자가 / 주소로 접속하면~" 이라는 뜻
▶ 예) www.naver.com/news

6 뷰 함수(View Function)

def hello():
  return "안녕하세요!"

▶ 사용자에게 보여줄 내용을 반환하는 함수

7 HTML 출력하기

render_template_string()

▶ HTML 코드를 문자열로 만들어서 반환

8 삼중따옴표 사용

'''
<html>...</html>
'''

▶ 여러 줄의 HTML을 편하게 작성하는 기술

9 HTML 기본 구조
<html>
  <head>
    <title>제목</title>
  </head>
  <body>
    내용
  </body>
</html>
10 제목 추가

<h1>안녕하세요!</h1>

▶ 가장 큰 크기의 제목 태그

11 문단 추가

<p>설명글...</p>

▶ 일반 텍스트를 표시하는 태그

12 스타일 적용

<body style="text-align: center;">

▶ 가운데 정렬처럼 간단한 디자인 적용

13 앱 실행 코드

if __name__ == '__main__':
  app.run(debug=True)

▶ "이 파일이 직접 실행될 때" 웹서버 시작

14 디버그 모드

debug=True

▶ 오류 발생 시 상세정보 표시
▶ 개발 중에만 사용 (배포 시 False)

15 파일 저장

▶ 파일명: app.py
▶ 인코딩: UTF-8 (한글 지원 필수!)

16 실행 방법

1. 터미널 열기
2. python app.py 입력
3. 서버 시작 메시지 확인

17 접속 확인

▶ 브라우저 주소창에:
http://localhost:5000

▶ "안녕하세요!" 페이지가 보여야 함

18 수정 후 재시작

▶ 코드 수정 → 서버 재시작
▶ Ctrl+C (중지) → 다시 실행
